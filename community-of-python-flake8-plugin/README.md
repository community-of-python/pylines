# community-of-python-flake8-plugin

Community of Python flake8 plugin with custom code style checks.

## Run with uv

```bash
uv run --with flake8 --with "community-of-python-flake8-plugin @ git+https://github.com/community-of-python/pylines.git#subdirectory=community-of-python-flake8-plugin" -- flake8 path/to/file.py
```
